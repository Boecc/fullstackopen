# Full Stack Open 2025
My solutions for the [Full Stack Open](https://fullstackopen.com/en/) course offered by the University of Helsinki.

This course serves as a deep dive into modern web development, covering JavaScript, React, Redux, Node.js, MongoDB, GraphQL, TypeScript, React Native, and CI/CD.
